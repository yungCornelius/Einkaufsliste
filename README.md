# Einkaufsliste
Studienleistung BSI II


Mit dieser App kannst du Artikel zu einer Einkaufsliste hinzufügen und durch Antippen wieder entfernen.

## Features
- Artikel hinzufügen
- Artikel anzeigen
- Artikel durch Antippen löschen
